# excel-sales-dashboard
Bu repository-də satış məlumatları əsasında özüm hazırladığım Excel dashboard layihəsi yer alır.
## Layihə haqqında
Bu layihəni Excel-də data analizi və dashboard hazırlama bacarıqlarımı inkişaf etdirmək və nümayiş etdirmək məqsədi ilə yaratmışam.
Layihədə:
- Satış və profit analizi
- Məhsul və region üzrə müqayisələr
- Discount-ların satışlara təsiri
- Pivot table və qrafiklərdən istifadə olunub.
## İstifadə etdiyim alətlər
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel formulaları
## Məqsəd
Excel data analizi və dashboard hazırlama bacarıqlarımı portfolio üçün göstərmək.
